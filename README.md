# AI-Driven Multimodal Diabetes Prediction & Intelligent Retinal Monitoring System

An intelligent healthcare analytics platform that integrates machine learning–based diabetes prediction with retinal image analysis to assist in early diabetes risk assessment and multimodal healthcare monitoring.

---

# Key Features

* Real-time diabetes risk prediction using machine learning
* Retinal image upload and analysis workflow
* Unified multimodal healthcare risk assessment
* AI-generated healthcare insights and recommendations
* Confidence score generation for prediction transparency
* Interactive Streamlit-based healthcare dashboard

---

# Technology Stack

* Python
* Scikit-learn
* TensorFlow
* Random Forest Classifier
* Streamlit
* NumPy
* Pandas

# Installation & Setup

## Clone Repository

```bash id="3v8qwm"
git clone https://github.com/your-username/multimodal-diabetes-assistant.git
```

---

## Navigate to Project Directory

```bash id="6m1xpl"
cd multimodal-diabetes-assistant
```

---

## Create Virtual Environment

```bash id="2q7vkn"
python -m venv venv
```

---

## Activate Virtual Environment

### Windows

```bash id="9m4wql"
venv\Scripts\activate
```

### macOS/Linux

```bash id="5x2vzm"
source venv/bin/activate
```

---

## Install Dependencies

```bash id="1p8qwm"
pip install -r requirements.txt
```

---

## Train Model

```bash id="4n7xvl"
python train_diabetes_model.py
```

---

## Run Application

```bash id="8q1wkn"
python -m streamlit run app.py
```
